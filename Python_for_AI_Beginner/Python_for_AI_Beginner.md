## 1.6 Data in Python

- **String**: "text"; triple"": for multi-line string
  
  ```python

  ```

- **Numbers**: Integers and Floats

- **Check type of data in python**
  
  ```python
  type("Hello, world!")
  # <class 'str'>
  type(100)
  # <class 'int'>
  type(2.99)
  # <class 'float'>
  type("2.99")
  # <class 'str'>
  ```

- Using Python as a calculator
  
  ```python
  print(2+9)
  print(57-8)
  print(12*9.2)

  print(1.05**10)
  ```
